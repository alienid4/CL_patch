# 多表結構快取（已撈資料，免重撈）

> 目的：這些是「開 Excel 掃一遍才知道」的事實，撈一次很花時間又常卡在檔案鎖。固化在這，下個 AI 直接用。
> 若懷疑過期，才重掃：先關 Excel → `cp` 一份 → `py` + openpyxl 讀。對映決策見 `docs/多表欄位對照表_草稿.md`。

## 檔案與操作事實
- 資料檔：`docs/弱點彙總報告(New)_V20260702 的TEST.xlsx`
- **常被 Excel 開著鎖住** → 直接讀會 PermissionError / 瀏覽器 fetch 404。先關 Excel，或先 `cp` 到暫存再讀。
- **這份是稀疏樣本**：大表(1/2/3/4/10)有幾百「列」但**有內容的只有 1–2 列**（其餘空白列）；小表(5–9)才有幾筆真資料。換完整資料檔對映照跑。
- 共 11 張表：10 張「數字-」開頭 + `資安弱點追蹤窗口`（項目→窗口對照，**不做分頁**）。
