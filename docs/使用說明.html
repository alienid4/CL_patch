弱點彙總 Dashboard

把「弱點彙總報告 Excel」一鍵變成可追蹤、可催辦、可交主管的互動看板

i 這是什麼？解決什麼？

一個純網頁、免安裝、不上傳的小工具。用瀏覽器開啟，選一個 Excel，馬上得到弱點追蹤看板。

以前的痛：每次要人工在 Excel 篩「資訊架構部 / 未結案」、算真正到期日（例外 → 展延 → 修補）、數逾期、找負責人、一封封寫催辦信……耗時又容易算錯。
現在：丟 Excel 進去，上面全部自動算好，還能點數字看明細、一鍵產生催辦信。
分析條件：資訊架構部 · 未結案
真正到期日：例外核准 → 首次展延 → 修補期限
檔案只在你電腦解析，不外傳

5 使用 SOP（5 步驟）

取得工具

向資安小組索取 CL_patch 資料夾（或共用路徑），整包複製到你電腦即可，無需安裝。

開啟

進入資料夾，用瀏覽器（Chrome / Edge）雙擊 index.html。

匯入 Excel

把「弱點彙總報告(New).xlsx」拖進畫面或按「選擇檔案」。系統自動解析（檔案不上傳，只在本機）。

看 & 追

四個分頁切換：Dashboard 看總覽、人員追蹤按「產生催辦內容」發信、例外統計盯治理、查詢找特定 CASE。任何數字都能點開明細。

下次更新

拿到新版 Excel，重開後直接匯入新檔即可覆蓋。不匯入新檔時，會自動記住上次資料，開啟即見。

! 重點提醒

  • 資料不外傳：Excel 只在你的瀏覽器本機解析，不會上傳任何伺服器。
  • 免安裝、可離線：一個資料夾、一個瀏覽器就能跑。
  • 記住上次匯入：存在「這台電腦、這個瀏覽器」，換機器不會跟著走；「🗑 清除記憶」可清除。
  • 依欄位名稱讀取：Excel 欄位順序改變仍可運作；缺必要欄位會明確提示。
